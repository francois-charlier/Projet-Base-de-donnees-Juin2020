Services :
-> Accueil ( permet de choisir si on s'incrit ou se connecte à notre user, page d'accueil du site. )

-> Connection ( permet de se connecter a son compte,  2 paramètres qui sont pseudo et mot de passe )

-> Inscription ( permet de se connecter a son compte,  7 paramètres qui sont nom, mot de passe, prenom, pseudo, date de naissance, sexe, mail)

-> Catégories sondages ( permet d'afficher les sondages selon la catégorie sélectionnée, 1 paramètre qui est la catégorie selectionnée )

-> Sondages ( permet d'ouvrir un sondage, 1 paramètre qui est l'id du sondage sélectionné )

-> Classement ( permet de classer les sondages selon le nombres de participants, pas de paramètre )

-> Profils ( permet d'afficher le profil des utilisateurs, 1 paramètre qui est l'id de l'utilisateur)
